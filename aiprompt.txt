
# Introduction
Download video, transcribe them in order to chat with the content later. The transscriptions for a group should be grouped in folders if they belong to different groups.

# Objective
1) Create a script (Powershell, Python or similar) than can download videos from youtube and vimeo and transcribe them into full texts
2) Download the videos below using yt-dl to videos folder. Add a gitignore to ignore the videos folder
3) Use OpenAI to transcribe the videos into the best possible Danish quality (I assume whisper API - via cloud and not local).
